INSTRUCTIONS
Le but du présent TD est de faire en équipes de 4 un site internet pour un
concours de photos ornithologiques dont le sujet est l’espace naturel de la motte.
Cet espace a déjà un site, mais il est nul :
• https://www.tourisme-seine-et-marne.fr/visiter-decouvrir/4686649-
espace-naturel-motte/
J’ai moi-même réalisé une page web similaire à ce que vous pourriez faire, vous
pouvez vous en inspirer :
• https://emmanuelrauzy.alwaysdata.net/Motte
Attention ! La page que j’ai faite ne contient qu’une page dédiée au concours photos,
il vous faudra au moins deux pages (accueil/espace de la Motte et concours photos).
Les membres d’une même équipe travaillent à différents endroits de la classe et
ne peuvent pas communiquer en parlant. (Pour simuler un travail à distance.)


instructions pour les dev :
Créer une page html d’accueil et une page php qui sert au concours photo.
• En utilisant au moins quatre classes CSS dont le nom est donné par le/la
cheffe d’équipe.
• Créer un formulaire pour soumettre les photos (qui sont enregistrées sur le
site) et un formulaire de vote.
